---

## Description

**Bangla Web Fonts** enables seamless integration of Bangla typography into your WordPress site. With proper font rendering across all modern devices and browsers, your Bangla text will appear just the way it does in printed textbooks—clean, elegant, and consistent.

Whether you're running a blog, news portal, or educational platform, this plugin ensures your Bangla content looks beautiful and professional.

---

## Features

- Automatically embeds Bangla web fonts site-wide.
- Uses the popular and readable **Kalpurush** typeface.
- Ensures compatibility across devices and browsers.
- Lightweight and easy to install—no configuration needed.

---

## Usage

Once the plugin is activated, it attempts to embed the Bangla font automatically.

**If it doesn’t take effect**, you can manually apply the font by editing your theme's CSS:

```css
body {
    font-family: 'Kalpurush', Nunito, sans-serif !important;
}

Installation

Method 1: From WordPress Dashboard

1. Go to Plugins > Add New
2. Search for Bangla Web Fonts
3. Click Install and then Activate
4. Done!


Method 2: Manual Upload

1. Download the plugin ZIP file
2. Navigate to Plugins > Add New > Upload Plugin
3. Upload the ZIP and activate
4. Done!
